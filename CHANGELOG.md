# Driver Pay App — Changelog

## v5.2.0

- Added Standard Weekly Rest Foundation.
- End Week now stores a Weekly Rest Candidate from the last completed Finish.
- Worked / OT can be replaced by a compact Weekly Rest card on the first post-End Week workday.
- Weekly Rest card shows 45h or 24h weekly start proposals with compact helper text.
- Added six-work-cycle priority rule for 24h reduced weekly rest.
- Added `from 9h rest` helper for reduced daily Start suggestions.
- Kept Pay Profiles and Pay Setup architecture unchanged.
- Kept standard Saturday/Sunday Off and Monday workflow for now.
